# shin-backing
This is a wrap around backing/mount for the shin armor. 
Basic layout was:
- Front padding
    - suede
    - green polyuerathane foam 1/4-1/2"
    - Suede
- Main Front
    - Suede
    - Batting
    - soft flannel
- Main Back
    - Suede
    - Batting
    - soft flannel
I used these [zippers in 7" format from YKK](https://www.ebay.com/itm/352572545032).
